Automatic Climate Sensitive Window

Project Overview
This project automatically controls window opening and closing based on environmental conditions using multiple sensors.

Sensors Used
- DHT11 Temperature and Humidity
- LM35 Temperature Sensor
- DS18B20 Temperature Sensor
- Rain Sensor
- Moisture Sensor
- Water Level Sensor
- MQ135 Air Quality Sensor
- LDR Light Sensor
- BMP180 Pressure Sensor
- Wind Sensor

 Features
- Automatically closes during rain
- Detects high wind conditions
- Controls based on temperature
- Monitors air quality
- Uses multiple sensor readings
